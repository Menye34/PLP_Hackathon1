Library Management System

The Library Management System was a project given in school in the Analysis and Design course by our lecturer, Dr. Patrick. We were asked to create a system that would enable librarians to manage their library.

The group number was Group 11, made up of 7 members.

In this project, my group members and I were able to come up with the following:

  • Functional requirements
  • Non-functional requirements
  • UI/UX design
  • Class diagram
  • Use case diagram
  • Activity diagram
  • Sequence diagram

The software development life cycle we used to implement this library system was the Incremental Model because various information was known before the implementation.
